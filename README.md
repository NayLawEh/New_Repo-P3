# Project 3
**Visualizing Data**

## Purpose
The purpose of this project uses Python to study important traffic patterns and how people drive for the Federal Highway Administration. We analyzed data from two big travel surveys, NHTS and NGSIM. With Python, we made graphs and ran simple computer simulations to show how cars move using the Intelligent Driver Model. The results help us see real trends and make better choices for planning roads. This study shows that using Python is faster and more accurate than old spreadsheet methods.
